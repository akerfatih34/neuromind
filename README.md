# 🧠 NeuroMind — Brain Training / Beyin Antrenmanı

Memorado tarzı, tamamen ücretsiz ve açık beyin antrenmanı PWA'sı.
Free, open, Memorado-style brain training PWA.

**Canlı / Live:** `https://KULLANICIADIN.github.io/neuromind/`

## Özellikler / Features
- 5 kategoride 11 oyun (N-Back dahil) • 11 games across 5 categories (incl. N-Back)
- Adaptif zorluk (Sv 1–10) + oyun sonu zorluk geri bildirimi • Adaptive difficulty + feedback
- 0–1000 Beyin Skoru, günlük antrenman, 16 rozet, seri • Brain Score, daily workout, badges, streaks
- Nefes egzersizli Relax bölümü • Relax section with breathing exercises
- Çoklu profil, dosya yedeği, Google girişli Firebase bulut sync • Profiles, backup, Google sign-in cloud sync
- TR/EN, çevrimdışı çalışır, telefona kurulur (PWA) • TR/EN, offline, installable

## Kurulum / Setup
Ayrıntılar için **KURULUM.md** dosyasına bak. Özet:
1. Bu repoyu GitHub Pages ile yayınla (Settings → Pages → main / root).
2. Telefonda adresi açıp "Ana Ekrana Ekle".
3. (İsteğe bağlı) Firebase + Google girişi ile cihazlar arası senkronizasyon.

Tek dosyalık uygulama: `index.html`. Derleme/bağımlılık yok.
